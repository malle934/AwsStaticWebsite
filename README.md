# 🚀 AWS Static Website with CloudFront (OAC)

## 📌 Project Overview
This project demonstrates how to host a **secure and scalable static website** using Amazon S3 and CloudFront with Origin Access Control (OAC).

The frontend of this website was generated using **Antigravity AI**, showcasing the integration of AI-powered development with cloud deployment.

---

## 🏗️ Architecture

User (Browser)
      ↓
CloudFront (CDN + HTTPS)
      ↓
Origin Access Control (OAC)
      ↓
Private S3 Bucket (Static Website Files)

---

## 🔐 Key Features

- ✅ Private S3 bucket (no public access)
- ✅ Secure access using Origin Access Control (OAC)
- ✅ CloudFront CDN for fast global delivery
- ✅ HTTPS enabled
- ✅ AI-generated frontend using Antigravity
- ✅ Scalable and production-ready architecture

---

## 🛠️ Technologies Used

- AWS S3 (Storage)
- AWS CloudFront (CDN)
- Origin Access Control (OAC)
- HTML, CSS, JavaScript
- Antigravity AI (Frontend generation)
